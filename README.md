# wasm-unic
An experiment of providing WebAssembly bindings for [rust-unic](https://github.com/open-i18n/rust-unic)

[![npm version](https://badge.fury.io/js/wasm-unic.svg)](https://badge.fury.io/js/wasm-unic)

## Build

```
wasm-pack build
```
